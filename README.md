## Quick Start

### Installation

```bash
# Clone repository
git clone https://github.com/yourusername/vajra-stream.git
cd vajra-stream

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Set up database
python scripts/setup_database.py
```

### Basic Usage

```bash
# Run a simple blessing (Level 2)
python scripts/run_blessing.py --intention "May all beings be happy" --duration 300

# With healing focus
python scripts/run_blessing.py --mode healing --chakra heart --duration 600

# Continuous background blessing
python scripts/run_blessing.py --continuous
```
